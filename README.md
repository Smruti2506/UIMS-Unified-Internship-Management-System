# 🎓 UIMS - Unified Internship Management System

The **Unified Internship Management System (UIMS)** is a full-stack web application developed as an MCA Final Year Project. It is designed to streamline and digitize the entire internship and live project management process for students, faculty, and administrators.

This system eliminates manual tracking by providing a centralized platform where students can submit project proposals, track progress, and receive feedback, while administrators can monitor and evaluate performance efficiently.

---

## 🚀 Key Features

### 👨‍🎓 Student Module

* Student Registration & Login
* Submit Internship / Project Proposal
* Weekly Progress Updates
* View Proposal Status (Approved/Rejected)
* View Marks & Feedback

### 🧑‍💼 Admin Module

* Secure Admin Login
* Review & Approve/Reject Proposals
* Monitor Student Progress
* Assign Marks & Feedback
* Dashboard Analytics (Progress Overview)

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Python (Flask)
* **Database:** MySQL / SQLite
* **Other Tools:** Tesseract OCR (for future enhancements)

## 🎯 Objectives

* To reduce manual workload for faculty
